# Measurement Lab Website

The official website for [Measurement Lab](https://measurementlab.net), built on the [Draftlab Scaffold](https://github.com/draftlab-org/scaffold).

## Development

```sh
npm install
npm run dev          # Start development server
npm run build        # Production build
npm run preview      # Preview production build
```

## Stack

- Astro v5 for static site generation
- Tailwind CSS v4 for styling
- React v19 for interactive components
- Pages CMS for content management

## Content Management

Content is managed through [Pages CMS](https://app.pagescms.org) with collections for pages, articles, people, navigation, and site configuration. All content lives in `src/content/` with type-safe schemas.
